# Học MSA Thực Chiến — GitHub Pages

Website đào tạo Measurement System Analysis bằng tiếng Việt cho đội ngũ QA/QC sản xuất cơ khí chính xác.

## Cập nhật repository

1. Giải nén gói ZIP.
2. Mở repository `tomsoft1608/hoc-msa-thuc-chien`.
3. Chọn **Add file → Upload files**.
4. Kéo toàn bộ nội dung đã giải nén vào thư mục gốc và xác nhận thay thế các file cũ.
5. Chọn **Commit changes**.

GitHub Pages sẽ tự triển khai lại từ branch `main`, thư mục `/(root)`.

## Chức năng

- 16 bài học chi tiết thuộc 4 module MSA.
- Câu hỏi tự kiểm tra riêng cho từng bài.
- Điều hướng bài trước/bài tiếp theo.
- Đánh dấu hoàn thành ở cuối bài và lưu tiến độ trên trình duyệt.
- Công cụ mô phỏng `%GRR` và `ndc`.
- Bài kiểm tra tổng hợp có chấm điểm.
